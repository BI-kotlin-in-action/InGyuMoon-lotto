# InGyuMoon-lotto
1. InputView 구현
   1. 로또 구입 금액을 입력 받는다.
      1. 로또는 1장에 천원이다
      2. 1000원 이하 금액은 버림
   2. 당첨 번호를 입력 받는다.
      1. 당첨 번호는 LottoGenerator를 통해 생성한다.
   3. 수동 로또의 개수를 입력받는다.
      1. 수동 로또 개수만큼 로또를 입력받는다.
      
2. OutputView 구현
   1. 구매한 로또를 오름차순으로 출력한다.
   2. 당첨 로또를 출력 한다.
   3. 총 상금을 출력한다.

3. LottoGenerator 구현
   1. 사용자가 구매한 로또 개수만큼 랜덤하게 로또를 발행해준다. 

4. Lotto 클래스 구현

5. Result 클래스 구현
   1. 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역을 구한다.
   2. 총상금을 계산한다.
